# 1. 快速上手：搭建 + 运行第一个项目

欢迎使用 NodePy！本教程将通过一个极简的项目，带你体验节点式编程的核心流程——**拖拽节点、连接、配置、运行、查看结果**。整个过程不需要写一行代码，只需用鼠标在画布上搭建一张“节点图”，就能完成数据读取、计算和可视化。

## 1.1 创建新项目

打开 NodePy 工作台，点击左侧的 **“新建项目”** 按钮，输入项目名称（例如“我的第一个项目”），进入节点编辑画布。

## 1.2 添加输入节点：手动创建一个简单的表格

我们要处理的数据从一个**表格**开始。  
- 在左侧节点菜单中，找到 **输入** 分类，将 **表格** 节点拖拽到画布上。  
- 双击该节点（或点击节点右上角的配置图标），打开参数配置面板。  

**配置表格内容**（模拟一份销售数据）：  
- **rows**：点击“添加行”，输入两行数据。例如：  
  第1行：`{"日期": "2024-01-01", "销量": 100, "单价": 20}`  
  第2行：`{"日期": "2024-01-02", "销量": 150, "单价": 22}`  
  第3行：`{"日期": "2024-01-03", "销量": 120, "单价": 21}`  
- **col_names**：输入列名列表 `["日期", "销量", "单价"]`（注意顺序要与 rows 中的键一致）。  
- **col_types**：指定每列类型。这里“日期”为 `"Datetime"`，“销量”和“单价”为 `"int"`。  

配置完成后点击“确定”，节点左上角会出现绿色小圆点，表示配置有效。此节点将输出一个包含三行三列的表格。

> 💡 **小贴士**：你也可以使用 **从文件上传** 节点读取本地的 CSV/Excel 文件，但为了快速上手，我们先用内置的表格节点。

## 1.3 添加计算节点：计算销售额

我们希望新增一列“销售额” = 销量 × 单价。这需要使用**列间运算**节点。  
- 在左侧菜单 **计算** 分类中，找到 **列间运算** 节点（即 `NumberColWithColBinOpNode`），拖拽到画布上。  
- 双击节点配置参数：  
  - **op**：选择 `MUL`（乘法）  
  - **col1**：选择 `销量`  
  - **col2**：选择 `单价`  
  - **result_col**：输入 `销售额`（如果不填，系统会自动生成列名）  
- 点击“确定”保存。

## 1.4 添加输出节点：绘制图表

为了直观看到销售额的变化，我们添加一个**快速绘图**节点。  
- 在左侧菜单 **可视化** 分类中，找到 **快速绘图** 节点，拖拽到画布上。  
- 配置图表：  
  - **x_col**：选择 `日期`（作为 X 轴）  
  - **y_col**：点击添加，选择 `销售额`  
  - **plot_type**：为 `销售额` 选择 `line`（折线图）  
  - 其他参数可暂留默认  
- 点击“确定”。

## 1.5 连接节点：搭建数据流

现在我们需要将三个节点连接起来，形成数据通路。  
- 将鼠标悬停在 **表格** 节点的右侧输出端口（`table`）上，会出现一个可拖拽的圆点。  
- 按住鼠标左键，拖出一条连线，连接到 **列间运算** 节点的左侧输入端口（`table`）。松开鼠标，连线建立。  
- 同样，将 **列间运算** 节点的右侧输出端口（`table`）连接到 **快速绘图** 节点的左侧输入端口（`input`）。  

此时画布上的节点图应该像这样（文字描述）：  
```
[表格] --(table)--> [列间运算] --(table)--> [快速绘图]
```

## 1.6 运行项目

每次节点图产生数据更新时系统会自动执行每个节点，并将数据传递下去。
当然，你也可以通过点击画布右下角的 **“同步”** 按钮，来手动触发节点执行。

如果一切正常，**快速绘图** 节点会生成一个图表文件。  
- 双击 **快速绘图** 节点，在右侧弹出的面板中可以看到生成的 PNG 图片，展示销售额随日期的折线图。  
- 你也可以点击图片下方的下载按钮，将图表保存到本地。

## 1.7 查看中间结果

如果想查看表格数据本身，可以双击 **列间运算** 节点，在输出面板中会显示当前节点的输出表格（包含新增的“销售额”列）。这有助于验证计算是否正确。

## 🎉 总结

通过这短短几步，你已经完成了第一个节点项目：
1. **输入**：用表格节点创建原始数据。
2. **处理**：用列间运算节点计算新字段。
3. **输出**：用快速绘图节点将结果可视化。
4. **连接**：用连线定义数据流向。
5. **运行**：一键执行，查看成果。

这就是节点编程的基本模式：**数据在节点间流动，每个节点完成一个特定的功能，组合起来就能实现复杂的数据处理任务**。接下来，我们将深入理解节点图的核心概念，以及更多常用节点的用法。
